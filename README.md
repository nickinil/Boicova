# Random Password Generator 🔐

**Автор:** Бойцова Рада Ивановна

## 📝 Описание проекта

Random Password Generator — это desktop-приложение на Python с графическим интерфейсом для генерации безопасных случайных паролей. Программа позволяет настраивать параметры пароля, оценивать его сложность, сохранять историю сгенерированных паролей и экспортировать данные в JSON.

### Основные возможности:

- ✅ **Генерация паролей** с использованием библиотеки `random`
- ✅ **Настройка параметров:**
  - Длина пароля (4-32 символа) с помощью ползунка
  - Выбор типов символов (строчные, заглавные, цифры, спецсимволы)
- ✅ **Оценка сложности** пароля в реальном времени
- ✅ **Сохранение истории** паролей в JSON с автоматической загрузкой
- ✅ **Копирование паролей** в буфер обмена
- ✅ **Управление историей** (просмотр, копирование, очистка)
- ✅ **Валидация ввода** (проверка длины и выбора символов)
- ✅ **Интуитивно понятный интерфейс**
